gxd-cli : 一种快速创建多容器工具

通过 docker run 命令行启动容器的时候，配置网络、挂载卷是一件非常麻烦的事，gxd-cli将这些麻烦的工作简化成修改配置文件TOML达成在不需要记忆繁琐的 docker 命令行参数就能快速启动多容器。

功能列表

  • 创建多容器，创建每个容器过程可以配置一下选项
    • 挂载卷（支持以pwd指代当前路径）
    • 指定容器的网络
    • 自定义容器名
    • 设定容器暴露的端口
  • 创建网络
  • 快速生成模板文件
