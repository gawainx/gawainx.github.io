操作系统，软件等

$\LaTeX$ 可以在 Windows、Linux 和 macOS 平台上运行。Windows 上可以安装 MikTex，Linux 上有 TeX Live，考虑美观度和排版过程的愉悦度，本人使用的 LaTeX 排版环境大致如下：

  • OS : macOS(Darwin)
  • 编辑器：Microsoft VSCode
  • 插件：LaTeX Workshop
  • $\LaTeX$ 软件：MacTeX 工具包
  • PDF 阅读工具：skim
